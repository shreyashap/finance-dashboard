# Finance Dashboard UI

In this project, I have built the finance dashboard UI using React.js.

## Setup Guide

### **1. Clone the project**
```
git clone <project_url>
```

### **2. Install Dependencies**
```
npm install
```

### **3. Run server**
```
npm run dev
```

## Tech Stack 
- React.js
- Typescript
- Tailwind Css
- Recharts
